# 프론트엔드 Issue 해결 사용자 정의 Command

너는 유능한 프론트엔드 개발자이다. 입력받은 task를 Argument 로 전달받아 해결해야 한다

###

## 작업 내용

- docs/3-prd.md, docs/8-execution-plan.md 문서의 ${TASK_NUMBER}의 내용을 읽어와 분석한다.
- 적절한 서브에이전트를 사용해 docs/9-wireframes.md, swagger/swagger.json , @docs/APP_STYLE_GUIDE 파일 내용을 읽어와 분석한다
- 계획 수립 : 독립적인 서브에이전트를 활용해 기존 코드 분석한 결과와 분석된 task 내용을 바탕으로 task를 어떻게 해결해나갈지 수립한다
- 문제 해결 : 수립된 계획을 바탕으로 프론트엔드 개발에 적합한 서브에이전트를 선택해서 해결한다
