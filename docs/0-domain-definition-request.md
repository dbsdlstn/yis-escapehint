너는 최고의 비즈니스 도메인 전문가야. "escapeHint"라는 이름의 방탈출 힌트제공 애플리케이션을 하려고 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 한눈에 들어오는 독창적인 UI 가 필요하다.
- 힌트를 관리자가 편하게 추가할 수 있어야하지만 해당 기능은 비밀스러워야한다.
- 힌트를 글로 제공해야 편하다.

# escapeHint 앱의 주요 기능 :

- 어떤 방탈출 테마인지 미리 설정하는 기능필요
- 관리자가 테마와 힌트를 추가하고 수정할 수 있어야함
- 관리자모드를 켜는 기능이 비밀스러워야함
- 고객은 힌트코드를 적는 칸, 줄어드는 플레이시간, 사용한 힌트수를 볼 수 있음
- 힌트 코드를 입력하면 힌트가 나오고 미리 설정된 진행률과 정답보기 버튼이 같이 있음
- 정답보기 버튼을 누르면 정답을 보여줘야함

# 필수 사항 :

- 도메인 정의서에는 핵심적인 내용만 포함하도록 함
- 작성된 도메인 정의서를 docs/ 디렉토리에 1-domain-definition.md 파일로 저장함
