## 1. 와이어프레임 개요

본 문서는 EscapeHint 시스템의 주요 화면에 대한 와이어프레임을 정의합니다. 와이어프레임은 텍스트와 ASCII 아트를 사용하여 각 화면의 레이아웃, 핵심 컴포넌트, 사용자 인터랙션 흐름을 시각적으로 표현합니다.

**디자인 원칙:**

- **플레이어 화면**: 몰입감 유지, 직관성, 가독성 (어두운 환경 고려)
- **관리자 화면**: 효율성, 명확성, 실수 방지

## 2. 플레이어 화면 와이어프레임 (Player-Facing)

- **기본 모드**: 다크 모드 (어두운 방에서의 눈부심 최소화)
- **주요 디바이스**: 태블릿 (가로 모드 기준), 모바일 (세로 모드)

### P-1: 테마 선택 화면

**설명**: 사용자가 처음 접속했을 때 보게 되는 화면입니다. 플레이할 테마를 선택합니다.

```
+------------------------------------------------------------------+
| 🔓 EscapeHint                                        [⚙️] [🌙]   |
|                                                                  |
|                "플레이할 테마를 선택하세요"                      |
|                                                                  |
| +------------------------+      +------------------------+       |
| |      좀비 연구소         |      |    시간 여행자의 방    |       |
| |                        |      |                        |       |
| |  "제한 시간: 60분"     |      |  "제한 시간: 90분"     |       |
| |  "난이도: ★★★★☆"     |      |  "난이도: ★★★☆☆"     |       |
| |                        |      |                        |       |
| |      [ 게임 시작 ]     |      |      [ 게임 시작 ]     |       |
| +------------------------+      +------------------------+       |
|                                                                  |
| +------------------------+                                       |
| |      해적선의 보물       |                                       |
| |  (Coming Soon)         |                                       |
| |  "제한 시간: 75분"     |                                       |
| |  "난이도: 미정"        |                                       |
| |                        |                                       |
| |      [ 준비중 ]        |                                       |
| +------------------------+                                       |
+------------------------------------------------------------------+
```

- **[⚙️]**: 관리자 모드 진입을 위한 숨겨진 버튼 영역 (예: 로고 5회 탭)
- **[🌙]**: 라이트/다크 모드 전환 토글
- **테마 카드**: 각 테마는 이름, 설명, 제한 시간, 난이도 등의 정보를 포함하는 카드로 표시됩니다.
- **상태 표시**: `Coming Soon` 이나 `준비중` 배지를 통해 아직 플레이할 수 없는 테마를 명확히 구분합니다.

---

### P-2: 메인 게임 화면 (힌트 입력)

**설명**: 게임이 시작된 후의 메인 화면입니다. 타이머, 힌트 입력 필드, 현재 진행 상태가 표시됩니다.

```
+------------------------------------------------------------------+
| 🧟 좀비 연구소                           ⏱️ 59:45                  |
|------------------------------------------------------------------|
|                                                                  |
|                                                                  |
|                  "힌트 코드를 입력하세요"                        |
|                                                                  |
|                       +------------------+                       |
|                       |  HINT01_         |                       |
|                       +------------------+                       |
|                                                                  |
|                           [ 확인 ]                               |
|                                                                  |
|                                                                  |
|------------------------------------------------------------------|
|  📊 진행률: 0%                💡 사용한 힌트: 0개                 |
+------------------------------------------------------------------+
```

- **상단 헤더**: 현재 플레이 중인 테마 이름과 실시간 타이머가 고정됩니다. 타이머는 00:00 도달 시 음수(-00:01)로 계속 카운트됩니다.
- **힌트 입력 필드**: 화면 중앙에 크게 위치하여 가독성과 조작성을 높입니다. 입력 시 자동으로 대문자로 변환됩니다.
- **하단 정보**: 현재 게임 진행률과 사용한 힌트 개수를 표시하여 플레이어가 진행 상황을 쉽게 파악할 수 있도록 합니다.

---

### P-3: 힌트 확인 화면

**설명**: 유효한 힌트 코드를 입력했을 때 표시되는 화면입니다. 힌트 내용과 함께 '정답 보기' 옵션을 제공합니다.

```
+------------------------------------------------------------------+
| 🧟 좀비 연구소                           ⏱️ 40:15                  |
|------------------------------------------------------------------|
|                                                                  |
|  [HINT01]                                                        |
|  "거울 뒤를 확인하세요. 숨겨진 숫자가 있습니다."                  |
|                                                                  |
|  진행률: ████░░░░░░░░░░░░░░░░ 15%                                  |
|                                                                  |
|                                                                  |
|                   [ 💡 정답 보기 ]                               |
|                                                                  |
|                   [ ← 돌아가기 ]                                 |
|                                                                  |
|------------------------------------------------------------------|
|  📊 진행률: 15%               💡 사용한 힌트: 1개                  |
+------------------------------------------------------------------+
```

- **힌트 내용**: 명확하고 큰 글씨로 힌트 내용을 보여줍니다.
- **진행률 바**: 해당 힌트를 확인했을 때의 게임 진행률을 시각적으로 보여줍니다.
- **정답 보기 버튼**: 정답을 바로 노출하지 않고, 사용자의 선택에 따라 확인할 수 있는 버튼을 제공합니다.
- **돌아가기 버튼**: 이전 게임 화면으로 돌아갑니다.

---

### P-4: 정답 확인 화면

**설명**: '정답 보기' 버튼을 클릭했을 때, 기존 힌트 내용 아래에 정답이 추가로 표시되는 화면입니다.

```
+------------------------------------------------------------------+
| 🧟 좀비 연구소                           ⏱️ 40:10                  |
|------------------------------------------------------------------|
|                                                                  |
|  [HINT01]                                                        |
|  "거울 뒤를 확인하세요. 숨겨진 숫자가 있습니다."                  |
|                                                                  |
|  +----------------------------------------------------------+    |
|  |  정답: 1234                                              |    |
|  +----------------------------------------------------------+    |
|                                                                  |
|                                                                  |
|                   [ ← 돌아가기 ]                                 |
|                                                                  |
|------------------------------------------------------------------|
|  📊 진행률: 15%               💡 사용한 힌트: 1개                  |
+------------------------------------------------------------------+
```

- **정답 표시**: 정답은 힌트와 명확히 구분되는 스타일(예: 박스, 배경색)로 표시됩니다.
- **돌아가기**: 정답 확인 후, '돌아가기' 버튼만 남아 화면 복잡도를 줄입니다.

---

### P-5: 게임 완료/실패 화면

**설명**: 게임이 종료되었을 때(성공, 시간 초과 실패 등) 표시되는 결과 화면입니다.

```
+------------------------------------------------------------------+
|                                                                  |
|                       🎉 축하합니다! 🎉                          |
|                       "좀비 연구소 탈출 성공"                    |
|                                                                  |
|      +----------------------------------------------------+      |
|      |  소요 시간: 52분 34초                              |      |
|      |  사용 힌트: 3개                                    |      |
|      |  최종 랭킹: 상위 35%                               |      |
|      +----------------------------------------------------+      |
|                                                                  |
|                                                                  |
|                     [ 다른 테마 선택하기 ]                       |
|                                                                  |
+------------------------------------------------------------------+
```

- **결과 요약**: 최종 소요 시간, 사용한 힌트 개수 등 핵심 결과 정보를 명확하게 보여줍니다.
- **다음 행동 유도**: '다른 테마 선택하기' 버튼을 통해 사용자가 자연스럽게 다음 플레이로 이어질 수 있도록 유도합니다.
- **실패 시**: "탈출 실패" 메시지와 함께 격려 문구를 표시합니다.

## 3. 관리자 화면 와이어프레임 (Admin-Facing)

- **기본 모드**: 라이트 모드 (밝은 환경에서의 가독성 우선)
- **주요 디바이스**: PC, 태블릿 (넓은 화면에 최적화)

### A-1: 관리자 로그인 모달

**설명**: 플레이어 화면의 특정 영역을 5회 탭하면 나타나는 관리자 로그인 모달입니다.

```
              +----------------------------------+
              |         🔐 관리자 로그인         |
              |----------------------------------|
              |                                  |
              |  비밀번호                        |
              |  +------------------------------+  |
              |  | **********_                  |  |
              |  +------------------------------+  |
              |                                  |
              |  [로그인]            [취소]      |
              |                                  |
              +----------------------------------+
```

- **보안**: 플레이어에게 노출되지 않는 방식으로 접근합니다.
- **단순함**: 비밀번호 입력과 로그인/취소 버튼만으로 구성하여 복잡도를 낮춥니다.

---

### A-2: 관리자 대시보드

**설명**: 관리자 로그인 후 처음 보게 되는 메인 화면입니다. 전체 현황을 요약하여 보여줍니다.

```
+-----------------+------------------------------------------------+
|  [EscapeHint]   |  👋 현우 님, 환영합니다.                       |
|-----------------|------------------------------------------------|
| 📊 대시보드     |                                                |
| 🎭 테마 관리    |  +--------------+  +--------------+  +-----------+
| 💡 힌트 관리    |  | 전체 테마    |  | 진행중인 세션|  | 오늘 사용된 |
| 📺 세션 모니터링|  |      5       |  |      3       |  | 힌트: 25회|
|                 |  +--------------+  +--------------+  +-----------+
|                 |                                                |
| 🚪 로그아웃     |  최근 활동 로그                                |
|                 |  +--------------------------------------------+ |
|                 |  | 15:30 | '좀비 연구소' 세션 시작됨          | |
|                 |  | 15:25 | 'HINT03' 힌트 수정됨             | |
|                 |  | 14:50 | '해적선' 테마 신규 등록            | |
|                 |  +--------------------------------------------+ |
+-----------------+------------------------------------------------+
```

- **사이드바**: 주요 기능(테마, 힌트, 세션 관리)으로 빠르게 이동할 수 있는 네비게이션을 제공합니다.
- **요약 카드**: 관리자가 가장 궁금해할 핵심 지표(테마 수, 현재 세션 수 등)를 상단에 배치합니다.

---

### A-3: 테마 관리 화면

**설명**: 모든 테마의 목록을 보고, 추가/수정/삭제/활성화 등 관리를 수행하는 화면입니다.

```
+-----------------+------------------------------------------------+
|  [EscapeHint]   |  🎭 테마 관리                                  |
|-----------------|------------------------------------------------|
| 📊 대시보드     |                               [+ 새 테마 등록] |
| 🎭 테마 관리    |  [전체(5)] [활성(4)] [비활성(1)]                 |
| 💡 힌트 관리    |------------------------------------------------|
| 📺 세션 모니터링| | 이름           | 제한시간 | 힌트 수 | 상태 | 관리 |
|                 |------------------------------------------------|
| 🚪 로그아웃     | | 좀비 연구소    | 60분     | 12개    | 🟢 | [...] |
|                 | | 시간 여행자    | 90분     | 8개     | 🟢 | [...] |
|                 | | 해적선의 보물  | 75분     | 10개    | 🟢 | [...] |
|                 | | 마법사의 탑    | 75분     | 15개    | 🔴 | [...] |
|                 | | (이하 생략)    |          |         |    |       |
+-----------------+------------------------------------------------+
```

- **테이블 뷰**: 테마 목록을 테이블 형태로 표시하여 정보를 한눈에 비교하기 쉽습니다.
- **상태 필터**: '활성'/'비활성' 필터로 원하는 테마만 쉽게 찾아볼 수 있습니다.
- **관리 메뉴 [...]**: 각 테마에 대한 [수정], [삭제], [힌트 관리] 등의 액션 메뉴를 제공합니다.

---

### A-4: 힌트 관리 화면

**설명**: 특정 테마에 속한 힌트들을 관리하는 화면입니다. 힌트 순서 조정이 핵심 기능입니다.

```
+-----------------+------------------------------------------------+
|  [EscapeHint]   |  💡 힌트 관리: 좀비 연구소                     |
|-----------------|------------------------------------------------|
| ...             |                        [+ 새 힌트 등록]        |
|                 |------------------------------------------------|
|                 | | ↕ | 코드   | 내용 (미리보기)    | 진행률 | 상태 | 관리 |
|                 |------------------------------------------------|
|                 | | ↕ | HINT01 | "거울 뒤를 확인..."| 10%    | 🟢 | [...] |
|                 | | ↕ | HINT02 | "책상 서랍 안쪽..."| 20%    | 🟢 | [...] |
|                 | | ↕ | HINT03 | "벽에 걸린 그림..."| 30%    | 🔴 | [...] |
|                 | | ... (이하 12개까지)                          |
|                 |------------------------------------------------|
|                 |                  * 힌트 순서를 드래그하여 변경 |
+-----------------+------------------------------------------------+
```

- **드래그 앤 드롭 [↕]**: 핸들을 드래그하여 힌트 순서를 직관적으로 변경할 수 있습니다.
- **내용 미리보기**: 힌트 전체 내용이 아닌 일부만 표시하여 테이블을 간결하게 유지합니다.
- **상태 표시**: 각 힌트의 활성/비활성 상태를 명확히 보여줍니다.

---

### A-5: 힌트 생성/수정 모달

**설명**: 새 힌트를 추가하거나 기존 힌트를 수정하기 위한 폼 모달입니다.

```
          +--------------------------------------------------+
          |                💡 새 힌트 등록                   |
          |--------------------------------------------------|
          |                                                  |
          |  힌트 코드*       진행률* (%)      순서(자동)     |
          |  +------------+   +------------+   +------------+ |
          |  | HINT04_    |   | 40         |   | 4          | |
          |  +------------+   +------------+   +------------+ |
          |                                                  |
          |  힌트 내용*                                      |
          |  +----------------------------------------------+ |
          |  | 금고의 비밀번호는...                         | |
          |  |                                              | |
          |  +----------------------------------------------+ |
          |                                                  |
          |  정답*                                           |
          |  +----------------------------------------------+ |
          |  | 8282                                         | |
          |  +----------------------------------------------+ |
          |                                                  |
          |  [저장]  [저장 후 다음 등록]  [취소]              |
          +--------------------------------------------------+
```

- **효율적 입력**: `저장 후 다음 등록` 버튼을 제공하여 여러 힌트를 연속으로 빠르게 등록할 수 있습니다.
- **필수 필드 표시**: `*` 표시로 필수 입력 항목을 명확하게 안내합니다.
- **자동 계산**: '순서'와 같이 시스템이 자동으로 정해줄 수 있는 값은 직접 입력하지 않도록 합니다.
